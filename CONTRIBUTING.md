# Contributing to MailO

Thank you for your interest in contributing to MailO! We welcome contributions from the community.

## How to Contribute

1. **Report Bugs**: Open an issue using the Bug Report template.
2. **Feature Requests**: Suggest new features via issues.
3. **Pull Requests**:
   - Fork the repository.
   - Create a feature branch.
   - Ensure your code follows the project's coding standards.
   - Submit a PR with a clear description of your changes.

## Coding Standards
- Follow C# coding conventions.
- Ensure all public methods are documented.
- Write unit tests for new functionality.

## Contact
Questions? Reach out to your-dev@yourdomain.com.
